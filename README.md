## ⬇️ Installation

1. Download the latest release from the **[Releases](https://github.com/apefart2222/Nitro-Menu/releases)** tab
2. Extract the `.dll` file
3. Place it in your Gorilla Tag `Plugins` folder  
4. Launch Gorilla Tag
