## ⬇️ Installation

1. Download the latest release from the **[Releases](https://github.com/yourusername/Nitro-Menu/releases)** tab
2. Extract the `.dll` file
3. Place it in your Gorilla Tag `Plugins` folder  
   (Usually: `Gorilla Tag/BepInEx/plugins/`)
4. **Disable your antivirus** temporarily if it flags the file (common with mod menus)
5. Launch Gorilla Tag

**How to open the menu:**
- Press **INSERT** / **F1** / **Right CTRL** (check in-game settings to change)
